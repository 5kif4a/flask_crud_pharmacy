Help system

  • Tab "Tables"

    This tab contains all the database tables.
    If you are an administrator, you can add, edit, delete data in tables.
    If in the process of modifying data integrity constraints are violated, "appears" warning messages.

  • Tab "Queries"

    This tab contains queries (reports) for a selection from the database.
    Each query has a description.

  • Tab "Views"

    This tab contains views (reports) of the database.
    Each view has a description.
    In the views of the data can not be edited.

  • Tab "Store procedures" (Only for Administrators)

    This tab contains stored procedures.
    Each procedure has a description.
    To call and execute a stored procedure, you need to select a procedure and set the required parameter.

  • Saving reports as PDF file

    You can save reports of tables, sample queries, views in the PDF format.
